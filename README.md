## Optional: Create Electron Development

```sh
git clone https://github.com/a-lang/keep.git
cd keep
npm install
```

## Optional: Build the distribution files

Build the application for specified platform:

```sh
npm start
npm run build:macos
npm run build:linux
npm run build:windows
```
